自己紹介

こんにちは。高専太郎といいます。鈴鹿高専電子情報工学科の１年生です。

生まれも育ちも三重県伊勢市です。

学校生活について

  • 授業
    • 高専なので、普通高校と違って実験や専門科目（私の場合は情報系学科なのでプログラミング）が多いです。授業時間も９０分と長いです。
    また、数学の進みが早く、難易度も高めです。試験前はみんなで集まって勉強したり、先生の研究室に質問に行ったりします。
    私はまだ１年生なので専門科目は少ないですが、高学年になるほど専門科目が増えていきます。

    難しかった授業トップ３

    1. 基礎数学A
      2. 覚悟はしてましたが、サインとかコサインがちょっと・・・。
      集合とかもあんま理解できてないです。

      三角関数はプログラミングでもよく使うと先輩からも聞いたので後期は頑張ります！

    2. 基礎数学B
      3. グラフを描いたり、グラフから解を求めたりするのが苦手です・・・。
      複素数もよく分からなかったです。

      後期はさらにグラフの内容が増えるっぽいので今から怯えてます＾＾；

    3. 国語IB
      4. 国語は中学の時からずっと苦手です・・・。特に古文・漢文は絶望しかないです。

      でも後期はないので一安心です！！

  • 部活
    • プロコンに所属しています。パソコン甲子園などの大会に向けて、日々プログラミングの勉強中です。
    先輩がめっちゃ優秀なので、一緒に活動していると勉強になることが本当に多いです！

    プログラミングスキルを高めたい人はぜひ！

    他にも運動系のクラブ・文化系のクラブ・ロボコン・デザコンなど、部活の種類は豊富です。
  • 行事
    • 鈴鹿高専には楽しい行事がたくさんあります！
    • 高専祭
      • 高専のイベントといったら真っ先に浮かぶのがこれです！
      私は昨年、友達と一緒に参加して、これで鈴鹿高専の受験を決めたくらい楽しかったです😆
      今年は展示をする側として参加する初めての高専祭なので、不安もありますが、それ以上に楽しみです！！
    • 研修旅行
      • 鈴鹿高専では以下の研修旅行があります。
      コロナの影響で、研修旅行に行けなかったり、行けても国内だった学年があったみたいです😢
      自分たちは全て行けることを祈ってます！
      • 新入生合宿研修
        • 曽爾高原で合宿を行いました！
        先生方や合宿施設の方々が色々なイベントを用意してくれています。
        おかげで、まだ入学したてで、ちょっと遠慮があったクラスメートとの距離が一気に縮まりました。
      • 学外研修（1年）・工場見学（3年 or 4年）
        • １年生の後期と、３年生または４年生のときと、計２回の学外研修旅行があります。
        １年生の学外研修は日帰り、３・４年生の工場見学は学科によって行き先が違い、
        日帰りか1泊2日かも学科によるみたいです。
        今から後期の学外研修がとても楽しみです😊
      • ２年生京都研修
        • 日帰りで京都市内を見学します。
        行き先は班ごとに決めるので、結構自由に見て回ることが出来るみたいです。
        すごく楽しみです！
      • ２年生海外研修
        • 台湾（元々はシンガポールだったみたいですがコロナの影響で・・・）に３泊４日で行きます。
        京都研修と同じく、班で自由に行動できる時間が結構あるみたいなので楽しみです！
        自分たちのときはシンガポールだったら嬉しいなあ。
    • 体育祭
      • １０月にあります。
      クラス内で誰がどの種目に出場するかを決めて、それぞれの種目でトーナメントを行うみたいです。
      最後にクラス対抗と学科対抗のリレーがあるらしいです。
      クラスTシャツを作りはしますが、先生や先輩から聞く限りだと、そこまで大規模なイベントって感じではなさそうです。
      体育祭をあまり重視しないのは高専らしいかも😅私は運動が苦手なので、ちょっと憂鬱です・・・。

地元のおすすめスポットランキング

  1. 伊勢神宮
    2. 当然の１位という感じですね😅
    伊勢というより、三重県を代表する観光スポット（と言って良いのかな？）だと思います。
    内宮と外宮があり、おかげ横丁があるためか、内宮の方が賑わっています。
    おかげ横丁は色々なお店があって楽しいです😆
    ただ休日とかはめっちゃ混雑してるので、静かに落ち着いて参拝したい人は外宮のほうがおすすめです！
    （私は外宮の方が好きです😊）

  2. 朝熊山展望台

    3. 自分の中での2位はここですね。
    ここに行くまでの伊勢志摩e-POWER ROADは有料道路ですが、
    道幅の広いワインディングで、景色も良いので車で走ると気持ちよさそうです。
    ここに来ると父はいつも楽しそうに運転してますが、スピードを出しすぎてよく母に怒られてます😅
    休日はスポーツカーが集まるイベントが開催されていることもあり、車好きの人には特におすすめです！
    自分もいつかスポーツカーでここを走ってみたいです😆
    山頂の展望台では食事も出来るし、お土産も売っています。
    ここの伊勢うどんと天空ソフトクリームは絶品なので、ぜひとも食べてみて下さい！
    天空のポストもあります！
    景色もとてもきれいです！空気が澄んでいるときなら知多半島もよく見えます。

  3. 夫婦岩
    4. 第3位は夫婦岩ですね。実は行ったことないですが😅
    地元の有名観光スポットなので挙げさせていただきました。
    江戸時代は以下のルートで参拝するのが正式だったようです。

    夫婦岩（二見）→ 伊勢神宮外宮（豊受大神宮）→ 伊勢神宮内宮（皇大神宮）→ 朝熊岳

    地元民ですが全然知りませんでした😓そもそも行ったことないですからね・・・。
    来年の初詣ではこの正式ルートで参拝したいです！
